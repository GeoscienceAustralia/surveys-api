Surveys

The survey Register, linked to above right, is the listing of all Surveys. It can be accessed directly using the URI:

  • http://pid.geoscience.gov.au/survey/ (not implemented yet)

Additionally, individual surveys may be accessed using a URI of the following form:

  • http://pid.geoscience.gov.au/survey/{SURVEY_NO} -- the {SURVEY_NO} is an integer number for a particular survey
