Surveys Linked Data API

About

This home page of the GA Samples Linked Data API is intended to provide basic information about this API (or 'Web Service'). In addition to this page, here are other sources of information about this service:

  • this service's entry in the main Geoscience Australia catalogue, eCat
  • an OGC, WMS web service standard GetCapabilities-like XML descriptor: {{ url_for('routes.index') }}?_view=getcapabilities&_format=application/xml
  • this system's git code repository

Surveys

The survey Register, linked to above right, is the listing of all Surveys. It can be accessed directly using the URI:

  • http://pid.geoscience.gov.au/survey/

Additionally, individual surveys may be accessed using a URI of the following form:

  • http://pid.geoscience.gov.au/survey/{SURVEY_NO} -- the {SURVEY_NO} is an integer number for a particular survey
